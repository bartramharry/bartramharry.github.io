Mathematical Calculations

Basic arthmetric

Arithmetic is used all the time in computer programs, so an understanding of how computers use arithmetic is useful. The adjacent table lists the common arithmetic processes with their programming equivalents:


Arithmetic process Programming equivalent
Addition (plus) +
Subtraction (minus) -
Multiplication *
Division /

Simple calculations

Computers can be programmed to carry out calculations so long as the correct formulas to use are known. Some simple mathematical examples in pseudocode are:

  • print (4 + 5) 9
  • print (4 * 6) 24
  • print (35 / 7) 5
  • print (20 - 7) 13
