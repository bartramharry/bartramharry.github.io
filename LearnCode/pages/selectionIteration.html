Selection and Iteration

Iteration

Iteration is the process of looping or repeating sections of a program. There are two types of iteration count controlled or condition controlled, count controlled loops repeat the same process a specific number of times. condition controlled loops repeat the same process over and over untill it gets a specific result.

In programming, iteration is often referred to as ‘looping’, because when a program iterates it ‘loops’ to an earlier step

Count controlled loops

A count-controlled loop is so called because it uses a counter to keep track of how many times the algorithm has iterated.

So for example an F1 race may have 10 laps, while the number of laps is less than (<) 10 the drivers will continue the loop, once it reaches 10 the loop with finish. It is a count controlled loop because the number of iterations to occur is already known.

These are known as FOR loops in code and are used to repeat a specific code block a known number of times.

Condition controlled loops

A condition-controlled loop is so called because iteration continues while, or until, a condition is met.

An example for this would be a race where the drivers race while they still have fuel, when fuel runs out they return to the pitstop, this is know as a WHILE loop in coding since it's not specifically known how many laps a driver will do until they need fuel. so WHILE they have fuel the race continues.

In programming, condition-controlled loops are implemented using WHILE statements.

Selection

Selection is a decision or question.
At some point, a program may need to ask a question because it has reached a step where one or more options are available. Depending on the answer given, the program will follow a certain step and ignore the others.

Many solutiuons feature several choices or decisions, these are know as IF and ELSE statements.
An example would be queueing for a rollercoaster IF you have a fast pass ticket you queue in line A ELSE you queue in line B.

IF statements

Programs consist of a set of instructions that are carried out one after another. Sometimes there may be more than one path (or set of steps) that can be followed. At this point, a decision needs to be made. This decision is known as selection. For example, this simple algorithm prints out a message depending on how old you are:
  • Ask how old you are
  • IF you are 18 or older
  • PRINT "You can learn to drive!"
